# Fragments

Aplicación Android que muestra el uso de fragments.

## Descripción

La aplicación está compuesta por una Activity principal que contiene dos fragments:

- Fragment de lista de contactos
- Fragment de detalle del contacto seleccionado

## Funcionamiento

Al seleccionar un contacto en la lista, se muestra su información en el fragment de detalle.

## Tecnologías

- Android Studio
- Kotlin
- XML

## Autor

Proyecto individual para práctica de fragments
